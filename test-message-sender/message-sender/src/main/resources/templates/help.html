How To Record a Script

  1. Upload a MAP or ProcessedMap file in ODE JSON format. The file should only contain a single MAP/ProcessedMap.
  2. Upload a SPAT or ProcessedSpat file in line-delimited ODE JSON format.
  3. Provide BSM data by either:
    • Manually placing a series of BSMs on the map using the draw controls (upper left on the map), or
    • Uploading a file containing BSMs in line-delimited ODE JSON format
  4. Manually created BSMs can be downloaded in line-delimited JSON format using the "Download BSMs" button, so that they can be re-used.
  5. Check the desired SPAT phases to send.
  6. Click Start under "Record Scripts"
  7. Click Start under "Send MAP", adjusting the send interval as desired.
  8. Click Start under "Send Selected SPAT Phases". Select "Loop", if desired, to repeat the series of SPATs indefinitely.
  9. Click Start under "Send BSMs". Adjust the send interval, or uncheck "fixed interval" to use the original BSM intervals if desired.
  10. When the BSMs and/or SPATs have finished being sent, as shown by the green background color turning off, click Stop under "Record Script"
  11. Click Download under "Record Script". The script can be run again using the script runner command line tool.
  12. Click Clear under "Record Script" to clear the script from memory. If the script is not cleared, subsequent recordings will append data to it.
